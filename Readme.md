# Base structure to make landing pages or static pages

This project use:
* Gulp
* Gulp SASS
* Gulp Concat
* Gulp Prefixer
* Gulp HTML Min
* Gulp Image Min
* Gulp Inject Partials
* Gulp Uglify
* Browser Sync
* Bootstrap SASS
* jQuery

## How use it?

To start to use it you need is had installed NodeJS and NPM or Yarn.

Once you've cloned or downloaded this project in your computer execute this command
```bash
npm install
```
or this one if you're using yarn
```bash
yarn install
```

Now you have all that you need to start to make static sites. To run the development server execute this command
```bash
gulp
```

If you want to use the Bootstrap JS execute
```bash
gulp vendors
```
